Impressoras de Rede (ESC/POS)

Imprima os pedidos direto em impressoras térmicas ligadas por Wi-Fi ou cabo (rede local). Requer o pequeno servidor-ponte "print-bridge.js" rodando no computador do caixa/cozinha.

Use "http://localhost:9123" se o servidor-ponte estiver rodando no mesmo computador que está com este painel aberto. Se estiver em outro computador da rede, use o IP dele (ex: http://192.168.0.10:9123).

Nova impressora

Como funciona

1. Baixe o arquivo print-bridge.js e rode node print-bridge.js no computador que está na mesma rede das impressoras.
2. Cadastre o IP de cada impressora (Wi-Fi ou cabo) — normalmente a porta é 9100.
3. Clique em "Imprimir" em qualquer pedido para enviar direto pra impressora, em formato ESC/POS.
